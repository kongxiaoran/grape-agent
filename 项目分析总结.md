## 关键概念速查

### Agent 执行循环
```python
while step < max_steps:
    check_cancellation()
    summarize_if_needed()
    response = llm.generate(messages, tools)
    if no_tool_calls(response):
        return response
    execute_tools(response.tool_calls)
    step += 1
```

### Progressive Disclosure Skills
```
Level 1: 元数据 (系统提示注入)
   ↓
Level 2: 完整内容 (get_skill 工具)
   ↓
Level 3: 资源文件 (按需读取)
```

### MCP 协议
```
Agent (Client) → MCP Server → External Service
支持: STDIO / SSE / HTTP
```
