## 📊 技术深度剖析

### 核心技术点（文档详细讲解）

#### 1. 五层架构设计
```
CLI 交互层 (cli.py)
    ↓
Agent 核心层 (agent.py)
    ↓
工具系统层 (tools/)
    ↓
LLM 客户端层 (llm/)
    ↓
外部服务层 (API/MCP/文件系统)
```

#### 2. Agent 执行循环（核心中的核心）
```python
async def run(self):
    while step < max_steps:
        # ① 检查取消
        if self._check_cancelled():
            return "Task cancelled"
        
        # ② Token 摘要
        await self._summarize_messages()
        
        # ③ 调用 LLM
        response = await self.llm.generate(messages, tools)
        
        # ④ 执行工具
        for tool_call in response.tool_calls:
            result = await tool.execute(**arguments)
        
        # ⑤ 判断完成
        if not response.tool_calls:
            return response.content
```

#### 3. Token 溢出自动摘要
- **触发条件**：本地估算 > 80K OR API 报告 > 80K
- **策略**：保留所有 user 消息，压缩 agent 执行过程
- **效果**：3000 tokens → 800 tokens (节省73%)

#### 4. Progressive Disclosure Skills
```
Level 1: Metadata (启动时加载)
    ~500 tokens
    ↓ Agent 决定使用
Level 2: Full Content (按需加载)
    ~2000-5000 tokens
    ↓ Skill 引用资源
Level 3: Resources (使用时加载)
    按需读取文件
```

#### 5. 工具系统设计
```python

    @property
    def name(self) -> str: ...
    
    @property
    def description(self) -> str: ...
    
    @property
```
